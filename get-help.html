If you are in immediate danger, call 911.
If you believe a crime is occurring, contact local law enforcement. HPRA does not provide emergency services.

First steps if you’re being targeted

  1. Preserve evidence: Take dated screenshots, save URLs, export platform data. Don’t engage with the abuser.
  2. Secure accounts: Change passwords, enable MFA, review recovery emails/phones, revoke suspicious sessions.
  3. Device check: Update OS/apps, run reputable AV, review installed apps/extensions; consider a fresh browser profile.
  4. Lock down data: Remove personal info from public profiles; consider data‑broker opt‑outs.
  5. Report & escalate: Use platform reporting; if threats are explicit, make a police report and request an incident number.

Contact HPRA

For non‑emergency support and referrals:

Email: support@humanprivacyrightsalliance.org

Documentation tips

  • Keep a log with dates/times, handles, and links.
  • Download full message threads when possible.
  • Preserve original files; avoid altering metadata.
